# SauceDemo Automation Framework

A professional Selenium-Java Automation Framework using the **Page Object Model (POM)** design pattern.

## 🚀 Features
* **Design Pattern**: Page Object Model (POM)
* **Reporting**: Extent Reports with automated screenshots on Success/Failure
* **Browser Management**: WebDriverManager with custom ChromeOptions
* **Testing Engine**: TestNG with Listeners
* **Build Tool**: Maven

## 🛠️ Setup
1. Clone the repository.
2. Ensure Maven and JDK 17 are installed.
3. Run the following command:
